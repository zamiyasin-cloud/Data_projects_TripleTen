# 📊 Superstore Sales Performance Dashboard

## Overview
Analyzed retail sales data to identify revenue trends, profit drivers, and underperforming segments.

## Tools Used
- Tableau
- Excel
- Data Cleaning & Aggregation

## Key Insights
- Identified top-performing product categories driving 60%+ of revenue
- Detected underperforming segments with negative profit margins
- Built KPI dashboards to track sales, profit, and regional performance

## Business Impact
Provided data-driven recommendations to improve profitability and optimize inventory allocation.
